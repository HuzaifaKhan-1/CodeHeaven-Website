JavaScript Variables

JavaScript variables are used to store data values. You can declare variables using var, let, or const.

let name = "John";
const age = 30;
var city = "New York";

JavaScript Data Types

JavaScript has different data types including strings, numbers, booleans, objects, arrays, and more.

let str = "Hello"; // String
let num = 100; // Number
let isTrue = true; // Boolean
let person = { name: "Alice", age: 25 }; // Object
let numbers = [1, 2, 3]; // Array

JavaScript Functions

Functions in JavaScript allow you to encapsulate code into reusable blocks. Functions can be declared or defined as expressions.

function greet(name) {
    return "Hello, " + name;
}
const add = (a, b) => a + b;

JavaScript Conditions

Conditional statements are used to perform different actions based on different conditions.

if (age > 18) {
    console.log("Adult");
} else {
    console.log("Not an Adult");
}

JavaScript Loops

Loops are used to repeatedly run a block of code until a certain condition is met.

for (let i = 0; i < 5; i++) {
    console.log(i);
}

let j = 0;
while (j < 5) {
    console.log(j);
    j++;
}

JavaScript Events

Events are actions that can be detected by JavaScript, such as clicks, key presses, or mouse movements.

document.getElementById("myButton").addEventListener("click", function() {
    alert("Button clicked!");
});

JavaScript Objects

Objects are collections of related data and functionalities. They are written as key-value pairs.

let person = {
    firstName: "John",
    lastName: "Doe",
    age: 30,
    greet: function() {
        return "Hello, " + this.firstName;
    }
};

console.log(person.greet());

JavaScript Arrays

Arrays are used to store multiple values in a single variable. Each value in an array is called an element.

let fruits = ["Apple", "Banana", "Cherry"];
console.log(fruits[0]);

fruits.push("Orange");
console.log(fruits);

JavaScript DOM Manipulation

The Document Object Model (DOM) represents the structure of a web page. JavaScript can be used to manipulate the DOM to change content, structure, and styling.

document.getElementById("myElement").innerText = "New Content";
document.getElementById("myElement").style.color = "blue";

JavaScript JSON

JSON (JavaScript Object Notation) is a format for storing and transporting data. It is often used when data is sent from a server to a web page.

let jsonString = '{"name": "John", "age": 30}';
let user = JSON.parse(jsonString);

console.log(user.name);

JavaScript Fetch API

The Fetch API is used to make network requests to retrieve resources from a server.

fetch('https://api.example.com/data')
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error('Error:', error));
