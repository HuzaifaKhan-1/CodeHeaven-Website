Introduction to Java

Java is a versatile and powerful programming language widely used for developing mobile applications, web applications, enterprise software, and more.

Java Installation

Java can be installed by downloading the JDK (Java Development Kit) from Oracle's website.

  1. Download the JDK from the official website.
  2. Install the JDK and set up the JAVA_HOME environment variable.
  3. Use an IDE like IntelliJ IDEA, Eclipse, or a text editor like VSCode.

Java Syntax

Java syntax is influenced by C and C++, but it has a simpler object model and fewer low-level facilities.


// Java Syntax Example
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, CodeHeaven!");
    }
}

Variables in Java

Variables in Java are declared with a specific data type.


// Variables Example
int x = 10;
String y = "CodeHeaven";
double z = 3.14;

Java Data Types

Java supports various data types such as integers, floating-point numbers, characters, and more.

  • int: Integer numbers
  • float: Floating-point numbers
  • char: Single character
  • String: Textual data

Java Operators

Operators in Java are special symbols used to perform operations on variables and values.

  • Arithmetic Operators: +, -, *, /, %
  • Relational Operators: ==, !=, >, <, >=, <=
  • Logical Operators: &&, ||, !
  • Assignment Operators: =, +=, -=, *=, /=, %=
  • Unary Operators: ++, --

Java Control Flow Statements

Control flow statements in Java determine the order in which the code is executed.

  • if-else: Conditional branching
  • switch-case: Multi-way branching

Loops in Java

Loops allow you to repeatedly execute a block of code as long as a condition is true.

  • for: Used when the number of iterations is known
  • while: Used when the number of iterations is unknown
  • do-while: Executes at least once, then checks the condition

Methods in Java

Methods are blocks of code that perform a specific task. They can accept parameters and return values.


// Method Example
public static int add(int a, int b) {
    return a + b;
}

Classes & Objects in Java

Classes are blueprints for creating objects. Objects are instances of classes.


// Class & Object Example
class Car {
    String model;
    int year;

    void display() {
        System.out.println("Model: " + model + ", Year: " + year);
    }
}

public class Main {
    public static void main(String[] args) {
        Car myCar = new Car();
        myCar.model = "Tesla";
        myCar.year = 2024;
        myCar.display();
    }
}

Constructors in Java

Constructors are special methods used to initialize objects. They have the same name as the class and no return type.


// Constructor Example
class Dog {
    String breed;

    // Constructor
    Dog(String breed) {
        this.breed = breed;
    }

    void display() {
        System.out.println("Breed: " + breed);
    }
}

public class Main {
    public static void main(String[] args) {
        Dog myDog = new Dog("Bulldog");
        myDog.display();
    }
}

Inheritance in Java

Inheritance allows one class to inherit properties and methods from another class.


// Inheritance Example
class Animal {
    void makeSound() {
        System.out.println("Animal sound");
    }
}

class Dog extends Animal {
    void makeSound() {
        System.out.println("Bark");
    }
}

public class Main {
    public static void main(String[] args) {
        Dog dog = new Dog();
        dog.makeSound();
    }
}

Polymorphism in Java

Polymorphism allows methods to perform different tasks based on the object that is invoking them.


// Polymorphism Example
class Animal {
    void sound() {
        System.out.println("Animal sound");
    }
}

class Cat extends Animal {
    void sound() {
        System.out.println("Meow");
    }
}

public class Main {
    public static void main(String[] args) {
        Animal animal = new Cat();
        animal.sound();  // Output: Meow
    }
}

Abstraction in Java

Abstraction is the concept of hiding the implementation details and showing only the essential features.


// Abstraction Example
abstract class Shape {
    abstract void draw();
}

class Circle extends Shape {
    void draw() {
        System.out.println("Drawing Circle");
    }
}

public class Main {
    public static void main(String[] args) {
        Shape shape = new Circle();
        shape.draw();
    }
}

Encapsulation in Java

Encapsulation is the concept of wrapping data and code together into a single unit, typically using access modifiers like private, public, etc.


// Encapsulation Example
class Student {
    private String name;

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }
}

public class Main {
    public static void main(String[] args) {
        Student student = new Student();
        student.setName("Huzaifa");
        System.out.println(student.getName());
    }
}

Interfaces in Java

Interfaces are used to achieve abstraction and multiple inheritance in Java.


// Interface Example

    }
}

public class Main {
    public static void main(String[] args) {

        dog.sound();
    }
}

Exception Handling in Java

Exception handling in Java is done using try, catch, and finally blocks.


// Exception Handling Example
public class Main {
    public static void main(String[] args) {
        try {
            int result = 10 / 0;
        } catch (ArithmeticException e) {
            System.out.println("Cannot divide by zero");
        } finally {
            System.out.println("Execution completed");
        }
    }
}

File Handling in Java

Java provides classes like File, FileReader, and FileWriter to handle files.


// File Handling Example
import java.io.File;
import java.io.FileWriter;
import java.io.IOException;

public class Main {
    public static void main(String[] args) {
        try {
            File file = new File("example.txt");
            if (file.createNewFile()) {
                System.out.println("File created: " + file.getName());
            }
            FileWriter writer = new FileWriter(file);
            writer.write("Welcome to CodeHeaven");
            writer.close();
        } catch (IOException e) {
            System.out.println("An error occurred.");
            e.printStackTrace();
        }
    }
}

Collections Framework in Java

The Collections Framework provides a unified architecture for storing and manipulating collections of objects.


// Collections Example
import java.util.ArrayList;

public class Main {
    public static void main(String[] args) {
        ArrayList list = new ArrayList<>();
        list.add("Java");
        list.add("Python");
        list.add("C++");

        for (String language : list) {
            System.out.println(language);
        }
    }
}

Generics in Java

Generics allow types to be parameters when defining classes, interfaces, and methods.


// Generics Example
class Box {
    private T value;

    public void set(T value) {
        this.value = value;
    }

    public T get() {
        return value;
    }
}

public class Main {
    public static void main(String[] args) {
        Box integerBox = new Box<>();
        integerBox.set(10);
        System.out.println(integerBox.get());

        Box stringBox = new Box<>();
        stringBox.set("CodeHeaven");
        System.out.println(stringBox.get());
    }
}

Multithreading in Java

Multithreading allows multiple threads to run concurrently.


// Multithreading Example
class MyThread extends Thread {
    public void run() {
        for (int i = 1; i <= 5; i++) {
            System.out.println(i);
            try {
                Thread.sleep(1000);
            } catch (InterruptedException e) {
                System.out.println(e);
            }
        }
    }
}

public class Main {
    public static void main(String[] args) {
        MyThread thread = new MyThread();
        thread.start();
    }
}

Streams & File I/O

Java provides the Streams API to perform input and output operations. This API includes classes and interfaces for reading and writing data to files and other I/O sources.


        // Streams & File I/O Example
        try (BufferedReader reader = new BufferedReader(new FileReader("input.txt"))) {
            String line;
            while ((line = reader.readLine()) != null) {
                System.out.println(line);
            }
        } catch (IOException e) {
            e.printStackTrace();
        }

Lambda Expressions

Lambda expressions provide a clear and concise way to represent one method interface using an expression. They are commonly used in functional programming and stream operations.


        // Lambda Expressions Example
        List numbers = Arrays.asList(1, 2, 3, 4, 5);
        numbers.forEach(n -> System.out.println(n));

Annotations

Annotations in Java are a form of metadata that provides data about a program, but are not part of the program itself. They have no direct effect on the operation of the code they annotate.


        // Annotations Example
        @Override
        public String toString() {
            return "CodeHeaven";
        }

JUnit Testing

JUnit is a framework for writing and running unit tests in Java. It is used to ensure that individual units of source code work as intended.


        // JUnit Testing Example
        import org.junit.jupiter.api.Test;
        import static org.junit.jupiter.api.Assertions.assertEquals;
        
        public class CalculatorTest {
            @Test
            public void testAdd() {
                Calculator calc = new Calculator();
                assertEquals(5, calc.add(2, 3));
            }
        }
