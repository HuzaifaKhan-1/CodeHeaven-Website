Introduction to C#

C# is a modern, object-oriented programming language developed by Microsoft, widely used for developing desktop applications, web applications, and more.

C# Installation

C# can be installed by downloading the .NET SDK from Microsoft's website.

  1. Download the .NET SDK from the official website.
  2. Install the SDK and set up your development environment.
  3. Use an IDE like Visual Studio or Visual Studio Code.

C# Data Types

C# supports various data types such as integers, floating-point numbers, characters, and more.

  • int: Integer numbers
  • float: Floating-point numbers
  • char: Single character
  • string: Textual data

C# Operators

Operators in C# are special symbols used to perform operations on variables and values.

  • Arithmetic Operators: +, -, *, /, %
  • Relational Operators: ==, !=, >, <, >=, <=
  • Logical Operators: &&, ||, !
  • Assignment Operators: =, +=, -=, *=, /=, %=
  • Unary Operators: ++, --

C# Control Flow Statements

Control flow statements in C# determine the order in which statements are executed.

  • if: Conditional execution
  • else: Alternate execution
  • switch: Multiple conditions
  • for: Loops with a counter
  • while: Loops with a condition
  • do-while: Loops at least once
  • foreach: Loops through collections

Loops

Learn about various loop constructs in C# like for, while, and foreach.


        // Example of a for loop
        for (int i = 0; i < 10; i++) {
            Console.WriteLine(i);
        }
        
        // Example of a while loop
        int j = 0;
        while (j < 10) {
            Console.WriteLine(j);
            j++;
        }
        
        // Example of a foreach loop
        int[] numbers = {1, 2, 3, 4, 5};
        foreach (int number in numbers) {
            Console.WriteLine(number);
        }

Polymorphism

Abstraction

Understand abstraction and how it hides implementation details from the user.


        public abstract class Shape {
            public abstract double Area();
        }
        
        public class Rectangle : Shape {
            public double Width { get; set; }
            public double Height { get; set; }
        
            public override double Area() {
                return Width * Height;
            }
        }
        
        public class Program {
            public static void Main() {
                Rectangle rect = new Rectangle { Width = 5, Height = 10 };
                Console.WriteLine($"Area: {rect.Area()}");
            }
        }

Encapsulation

Learn about encapsulation and how it protects data by bundling data and methods.


        public class BankAccount {
            private decimal balance;
        
            public void Deposit(decimal amount) {
                if (amount > 0) {
                    balance += amount;
                }
            }
        
            public decimal GetBalance() {
                return balance;
            }
        }
        
        public class Program {
            public static void Main() {
                BankAccount account = new BankAccount();
                account.Deposit(1000);
                Console.WriteLine($"Balance: {account.GetBalance()}");
            }
        }

Interfaces

Explore interfaces and how they define methods that must be implemented by classes.

Multithreading

Learn how to perform multiple tasks simultaneously using multithreading in C#.


        using System.Threading;
        
        public class Program {
            public static void Main() {
                Thread thread1 = new Thread(() => Console.WriteLine("Thread 1 is running"));
                Thread thread2 = new Thread(() => Console.WriteLine("Thread 2 is running"));
        
                thread1.Start();
                thread2.Start();
        
                thread1.Join();
                thread2.Join();
        
                Console.WriteLine("All threads completed");
            }
        }

Async Programming

Understand how to use async and await for asynchronous programming in C#.


        using System;
        using System.Threading.Tasks;
        
        public class Program {
            public static async Task Main() {
                Task task1 = Task1();
                Task task2 = Task2();
        
                await Task.WhenAll(task1, task2);
        
                Console.WriteLine("All tasks completed");
            }
        
            static async Task Task1() {
                await Task.Delay(1000);
                Console.WriteLine("Task 1 completed");
            }
        
            static async Task Task2() {
                await Task.Delay(2000);
                Console.WriteLine("Task 2 completed");
            }
        }

Delegates & Events

Learn about delegates and events, which are used for event handling and callback methods.


        public delegate void Notify(string message);
        
        public class Publisher {
            public event Notify OnNotify;
        
            public void NotifySubscribers(string message) {
                OnNotify?.Invoke(message);
            }
        }
        
        public class Program {
            public static void Main() {
                Publisher publisher = new Publisher();
                publisher.OnNotify += msg => Console.WriteLine("Received message: " + msg);
        
                publisher.NotifySubscribers("Hello, Delegates and Events");
            }
        }

LINQ

Understand how Language Integrated Query (LINQ) can be used for querying collections in C#.


        using System;
        using System.Linq;
        
        public class Program {
            public static void Main() {
                int[] numbers = { 1, 2, 3, 4, 5 };
        
                var evenNumbers = from number in numbers
                                  where number % 2 == 0
                                  select number;
        
                foreach (var number in evenNumbers) {
                    Console.WriteLine(number);
                }
            }
        }

Attributes

Learn how to use attributes to add metadata to your code.


        using System;
        
        [AttributeUsage(AttributeTargets.Class)]
        public class DescriptionAttribute : Attribute {
            public string Description { get; }
            public DescriptionAttribute(string description) {
                Description = description;
            }
        }
        
        [Description("This class performs mathematical operations")]
        public class MathOperations {
            // Methods here
        }
        
        public class Program {
            public static void Main() {
                var attributes = typeof(MathOperations).GetCustomAttributes(typeof(DescriptionAttribute), false);
                if (attributes.Length > 0) {
                    var description = (DescriptionAttribute)attributes[0];
                    Console.WriteLine($"Description: {description.Description}");
                }
            }
        }

Exercises

Practice your C# skills with the following exercises:

  • Loops: Write a program that prints the numbers from 1 to 100 using a loop. Modify the program to print only the even numbers.
  • Methods: Create a method that takes two integers as parameters and returns their sum. Write a program that calls this method and displays the result.
  • Classes & Objects: Design a class `Student` with properties for `Name`, `Age`, and `Grade`. Create an object of this class, set its properties, and display the information.
  • Constructors: Implement a class `Book` with a constructor that initializes `Title` and `Author`. Create an object of this class and display the book's details.
  • Inheritance: Create a base class `Animal` with a method `MakeSound()`. Derive a class `Cat` from `Animal` that overrides the `MakeSound()` method. Write a program that demonstrates polymorphism using these classes.
  • Polymorphism: Define a base class `Shape` with a method `Area()`. Derive classes `Rectangle` and `Circle` from `Shape` and override the `Area()` method to calculate the area of the respective shapes. Write a program that uses these classes to compute areas.
  • Abstraction: Create an abstract class `Appliance` with an abstract method `TurnOn()`. Derive classes `WashingMachine` and `Refrigerator` from `Appliance` and implement the `TurnOn()` method. Demonstrate the use of these classes.
  • Encapsulation: Develop a class `Account` with private fields for balance and public methods for deposit and withdrawal. Write a program that uses this class to manage account transactions.
  • Interfaces: Define an interface `IDriveable` with a method `Drive()`. Implement this interface in a class `Car` and write a program to demonstrate how the `Drive()` method is used.
  • Namespaces: Create a namespace `Utilities` and within it, define a class `Helper` with a static method `CalculateSum()`. Write a program that uses this class and method.
  • Exception Handling: Write a program that handles potential division by zero errors using try-catch blocks. Prompt the user to input two numbers and divide them, catching any exceptions that may occur.
  • File Handling: Write a program that reads text from a file and counts the number of words. Ensure to handle any exceptions that may arise during file operations.
  • Collections: Create a `List` and add some numbers to it. Write a program to find the maximum and minimum values in the list.
  • Generics: Implement a generic class `Pair` that holds two values of type `T`. Write a program that creates instances of this class with different data types and demonstrates its usage.
  • Multithreading: Write a program that creates and starts two threads. Each thread should print a message with a delay. Ensure the main thread waits for both threads to complete before exiting.
  • Async Programming: Create an asynchronous method that performs a time-consuming operation (e.g., simulating a delay). Write a program that calls this method and waits for its completion.
  • Delegates & Events: Implement a delegate for a method that processes a message. Create an event that triggers this delegate. Write a program that subscribes to the event and demonstrates its use.
  • LINQ: Write a LINQ query to filter a list of integers to include only those greater than 10. Display the results.
  • Attributes: Create a custom attribute `DeveloperInfo` that stores the name and date of a developer. Apply this attribute to a class and write a program that reads and displays the attribute information.
  • Unit Testing: Write unit tests for a method that calculates the factorial of a number. Ensure to test edge cases and verify that the method behaves as expected.
