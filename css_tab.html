Introduction to CSS

CSS (Cascading Style Sheets) is a language used to describe the presentation of a document written in HTML. It allows you to style and layout web pages, including changing colors, fonts, and spacing.

body {
    background-color: #f4f4f4;
    color: #333;
}

CSS Syntax

The CSS syntax consists of a selector and a declaration block. The selector points to the HTML element you want to style. The declaration block contains one or more declarations separated by semicolons.

selector {
    property: value;
    property: value;
}

CSS Selectors

CSS selectors are used to select the HTML elements you want to style. Common selectors include element selectors, class selectors, ID selectors, and attribute selectors.

p {
    color: #333;
}

.class-name {
    background-color: #f0f0f0;
}

#id-name {
    font-size: 16px;
}

[type="text"] {
    border: 1px solid #ccc;
}

CSS Colors

CSS allows you to apply color to your HTML elements using color names, HEX, RGB, RGBA, HSL, and HSLA values.

h1 {
    color: red;
}

p {
    color: #00ff00;
}

div {
    color: rgb(0, 0, 255);
}

.box {
    background-color: hsla(120, 100%, 50%, 0.3);
}

CSS Backgrounds

CSS allows you to style backgrounds of elements using background-color, background-image, background-repeat, background-size, and background-position properties.

body {
    background-color: #f0f0f0;
    background-image: url('image.jpg');
    background-repeat: no-repeat;
    background-size: cover;
}

CSS Borders

CSS border properties allow you to define the border around elements with styles like border-width, border-style, and border-color.

div {
    border-width: 2px;
    border-style: solid;
    border-color: #333;
}

CSS Margins

CSS margins are used to create space outside of an element's border. You can specify margins individually or together for all sides.

p {
    margin: 20px;
}

div {
    margin-top: 10px;
    margin-right: 15px;
    margin-bottom: 20px;
    margin-left: 25px;
}

CSS Padding

CSS padding properties are used to create space inside an element, between the content and its border. Like margins, padding can be set for all sides or individually.

div {
    padding: 10px;
}

section {
    padding-top: 15px;
    padding-right: 20px;
    padding-bottom: 25px;
    padding-left: 30px;
}

CSS Height and Width

CSS properties for height and width allow you to control the size of an element. You can set values in pixels, percentages, or other units.

div {
    width: 100%;
    height: 200px;
}

CSS Box Model

The CSS box model describes the rectangular boxes generated for elements in the document tree. It includes margins, borders, padding, and the content itself.

div {
    margin: 20px;
    border: 2px solid #333;
    padding: 15px;
    width: 200px;
}

CSS Text

CSS text properties control the appearance of text within elements. This includes text alignment, text decoration, line height, and text transformation.

p {
    text-align: center;
    text-decoration: underline;
    line-height: 1.5;
    text-transform: uppercase;
}

CSS Fonts

CSS fonts allow you to set the font family, size, weight, and style of text. You can use system fonts or custom fonts using @font-face or web font services.

h1 {
    font-family: 'Arial', sans-serif;
    font-size: 2em;
    font-weight: bold;
}

CSS Icons

CSS allows you to style icons using font-based icons like Font Awesome or SVGs. You can set size, color, and other properties for icons.

i {
    font-size: 24px;
    color: #333;
}

CSS Links

CSS allows you to style links in different states, such as normal, visited, hover, and active.

a:link {
    color: blue;
}
a:visited {
    color: purple;
}
a:hover {
    color: red;
}
a:active {
    color: green;
}

CSS Lists

CSS provides various options to style lists, including changing list markers, adding custom markers, and adjusting padding and margins.

ul {
    list-style-type: circle;
}
ol {
    list-style-type: decimal-leading-zero;
}
ul.custom-markers {
    list-style: url('marker.png');
}

CSS Tables

CSS can be used to style tables, including borders, spacing, and even alternating row colors for readability.

table {
    width: 100%;
    border-collapse: collapse;
}
th, td {
    border: 1px solid #ddd;
    padding: 8px;
}
tr:nth-child(even) {
    background-color: #f2f2f2;
}
th {
    background-color: #4CAF50;
    color: white;
}

CSS Positioning

CSS positioning allows you to control the layout of elements on a page using properties such as static, relative, absolute, and fixed.

div {
    position: absolute;
    top: 50px;
    left: 100px;
}

CSS Z-index

The CSS z-index property controls the vertical stacking order of elements that overlap.

div {
    position: absolute;
    z-index: 10;
}

CSS Overflow

The CSS overflow property controls how content is handled that exceeds its container. Options include visible, hidden, scroll, and auto.

div {
    overflow: hidden;
}

CSS Float

The CSS float property is used for positioning and formatting content, allowing elements to float to the left or right of their container.

img {
    float: left;
    margin: 10px;
}

CSS Flexbox

CSS Flexbox is a layout model that allows you to design flexible and responsive layout structures without using floats or positioning.

.container {
    display: flex;
    justify-content: space-between;
}

CSS Grid

CSS Grid Layout is a two-dimensional layout system for the web, allowing you to control both rows and columns together.

.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 10px;
}

CSS Align

CSS alignment properties allow you to align items within a container along the horizontal or vertical axis.

.flex-container {
    align-items: center;
    justify-content: center;
}

CSS Combinators

CSS combinators combine multiple selectors, such as descendant, child, adjacent sibling, and general sibling combinators, to style elements.

div p {
    color: red;
}

CSS Pseudo-classes

CSS pseudo-classes select elements based on their state, such as :hover, :active, :nth-child, and more.

a:hover {
    color: blue;
}

CSS Pseudo-elements

CSS pseudo-elements style specific parts of an element, such as ::before, ::after, ::first-line, and more.

p::first-line {
    font-weight: bold;
}

CSS Opacity

The CSS opacity property controls the transparency of an element, ranging from 0 (completely transparent) to 1 (completely opaque).

img {
    opacity: 0.5;
}

CSS Transitions

CSS transitions allow you to change property values smoothly over a given duration.

button {
    transition: background-color 0.3s ease;
}

CSS Animations

CSS animations allow you to animate the transition of properties, using keyframes to control the stages of the animation.

@keyframes example {
    from { background-color: red; }
    to { background-color: yellow; }
}

CSS Media Queries

CSS media queries allow you to apply styles based on the device's characteristics, such as screen size, resolution, or orientation.

@media (max-width: 600px) {
    div {
        background-color: lightblue;
    }
}
