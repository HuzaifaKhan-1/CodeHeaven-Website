Introduction to C

C is a powerful general-purpose programming language. It is widely used in system software, application software, device drivers, embedded systems, and more.

Installing C

You can use various compilers like GCC to write and compile C programs.

  1. Install a compiler like GCC.
  2. Use an IDE like Code::Blocks, Dev-C++, or Visual Studio Code.
  3. Write your C code in a text editor or IDE.

C Syntax

C programs follow a specific syntax and structure. The basic building blocks are functions, variables, and statements.


// C Syntax Example
#include <stdio.h>

int main() {
    printf("Hello, CodeHeaven!\n");
    return 0;
}

Variables in C

Variables in C are used to store data that can be modified during program execution.


// Variables Example
int x = 10;
char y = 'A';
float z = 3.14;

C Data Types

C supports various data types including integers, floating-point numbers, characters, and more.

  • int: Integer numbers
  • float: Floating-point numbers
  • char: Single character
  • double: Double-precision floating-point

C Operators

Operators in C are used to perform operations on variables and values.

  • Arithmetic Operators: +, -, *, /, %
  • Relational Operators: ==, !=, >, <, >=, <=
  • Logical Operators: &&, ||, !
  • Assignment Operators: =, +=, -=, *=, /=, %=
  • Unary Operators: ++, --

C Control Flow Statements

Control flow statements in C determine the order in which the code is executed.

  • if-else: Conditional branching
  • switch-case: Multi-way branching

Loops in C

Loops allow you to repeatedly execute a block of code as long as a condition is true.

  • for: Used when the number of iterations is known
  • while: Used when the number of iterations is unknown
  • do-while: Executes at least once, then checks the condition

Functions in C

Functions are blocks of code that perform specific tasks. They help in code reusability and organization.


// Functions Example
int add(int a, int b) {
    return a + b;
}

Arrays in C

Arrays in C are used to store multiple values of the same type in a single variable.


// Arrays Example
int numbers[5] = {1, 2, 3, 4, 5};

Pointers in C

Pointers store the memory address of another variable.


// Pointers Example
int x = 10;
int *ptr = &x;

Structures in C

Structures are used to group variables of different data types together.


// Structures Example
struct Person {
    char name[50];
    int age;
    float salary;
};

File Handling in C

File handling in C allows you to read and write data to files on disk.


// File Handling Example
FILE *file = fopen("data.txt", "r");
if (file == NULL) {
    printf("Error opening file\n");
}
fclose(file);

Preprocessor in C

The C preprocessor handles directives for code preprocessing before compilation.


// Preprocessor Example
#define PI 3.14

Dynamic Memory Allocation in C

C provides functions to allocate memory dynamically during runtime.


// Dynamic Memory Allocation Example
int *ptr = (int *)malloc(5 * sizeof(int));
if (ptr == NULL) {
    printf("Memory allocation failed\n");
}
free(ptr);

Recursion in C

Recursion is a technique where a function calls itself to solve a problem.


    // Recursion Example
    int factorial(int n) {
        if (n == 0) {
            return 1;
        } else {
            return n * factorial(n - 1);
        }
    }

Enumerations in C

Enumerations are a user-defined data type that consists of integral constants.


    // Enumeration Example
    enum week { Sunday, Monday, Tuesday, Wednesday, Thursday, Friday, Saturday };
    enum week today;
    today = Wednesday;
