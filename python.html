Introduction to Python

Python is a powerful programming language that is easy to learn...

Python Installation

Python can be installed from the official website. It's available for Windows, macOS, and Linux.

  1. Go to the official Python website.
  2. Download the latest version of Python.
  3. Run the installer and follow the instructions.
  4. Ensure you add Python to your PATH during installation.

Python Syntax

Python syntax is designed to be clean and readable. Code blocks are defined by indentation rather than brackets or keywords.


# Python Syntax Example
def greet(name):
    print(f"Hello, {name}!")

greet("CodeHeaven")

Variables in Python

Variables in Python are created by assigning a value to a name using the `=` operator.


# Variables Example
x = 10
y = "CodeHeaven"
z = 3.14

Python Data Types

Python supports various data types such as integers, floating-point numbers, strings, lists, tuples, and dictionaries.

  • int: Integer numbers
  • float: Floating-point numbers
  • str: Textual data
  • list: Ordered collection of items
  • tuple: Immutable ordered collection of items
  • dict: Key-value pairs

Python Operators

Python supports various operators for arithmetic, comparison, logical operations, and more.

  • Arithmetic Operators: +, -, *, /, %
  • Comparison Operators: ==, !=, >, <, >=, <=
  • Logical Operators: and, or, not

Control Flow in Python

Python supports control flow using `if`, `elif`, and `else` statements, as well as loops like `for` and `while`.


# Control Flow Example
age = 18
if age >= 18:
    print("You are an adult.")
else:
    print("You are a minor.")

Python Functions

Functions in Python are defined using the `def` keyword. They help to organize code into reusable blocks.


# Function Example
def add(a, b):
    return a + b

result = add(5, 10)
print(result)

Popular Python Libraries

Python has a rich ecosystem of libraries for various tasks. Some popular ones include:

  • NumPy: For numerical computations
  • Pandas: For data manipulation and analysis
  • Matplotlib: For data visualization
  • TensorFlow: For machine learning
  • Requests: For making HTTP requests

Python Loops

Loops in Python allow you to execute a block of code multiple times. Python supports different loop types like for loops and while loops.


# Loops Example
for i in range(5):
    print(i)

count = 0
while count < 5:
    print(count)
    count += 1

Python Lists & Tuples

Lists and tuples are both used to store collections of data. The main difference is that lists are mutable while tuples are immutable.


# Lists & Tuples Example
my_list = [1, 2, 3, 4]
my_tuple = (1, 2, 3, 4)

my_list[0] = 10  # Lists are mutable
# my_tuple[0] = 10  # This would raise an error since tuples are immutable

Python Dictionaries & Sets

Dictionaries and sets are both used to store collections of data, but they work differently.


# Dictionaries & Sets Example
my_dict = {"name": "Alice", "age": 25}
my_set = {1, 2, 3, 4}

print(my_dict["name"])
my_set.add(5)

Python List Comprehensions

List comprehensions provide a concise way to create lists based on existing lists or other iterables.


# List Comprehension Example
squares = [x**2 for x in range(10)]
even_squares = [x**2 for x in range(10) if x % 2 == 0]

Python File Handling

File handling allows you to read, write, and manipulate files in Python.


# File Handling Example
with open("example.txt", "w") as file:
    file.write("Hello, CodeHeaven!")

with open("example.txt", "r") as file:
    content = file.read()
    print(content)

Python Error Handling

Error handling in Python uses try-except blocks to manage exceptions and prevent crashes.


# Error Handling Example
try:
    x = 1 / 0
except ZeroDivisionError:
    print("Cannot divide by zero!")
finally:
    print("This block always executes.")

Python Object-Oriented Programming (OOP)

OOP in Python allows you to model real-world entities using classes and objects.


# OOP Example
class Dog:
    def __init__(self, name, breed):
        self.name = name
        self.breed = breed

    def bark(self):
        print(f"{self.name} says woof!")

my_dog = Dog("Buddy", "Golden Retriever")
my_dog.bark()

Python Modules & Packages

Modules and packages help you organize and reuse code across different projects.


# Modules & Packages Example
import math

print(math.sqrt(16))

Working with JSON in Python

JSON (JavaScript Object Notation) is a popular format for data exchange. Python provides built-in support for working with JSON.


# JSON Example
import json

data = {"name": "CodeHeaven", "type": "website"}
json_str = json.dumps(data)
print(json_str)

parsed_data = json.loads(json_str)
print(parsed_data["name"])

Python and Databases

Python can interact with databases using libraries like SQLite, MySQL, and PostgreSQL.


# Database Example with SQLite
import sqlite3

conn = sqlite3.connect("example.db")
cursor = conn.cursor()
cursor.execute("CREATE TABLE IF NOT EXISTS users (id INTEGER PRIMARY KEY, name TEXT)")
cursor.execute("INSERT INTO users (name) VALUES ('Alice')")
conn.commit()
conn.close()

Python Web Scraping

Web scraping involves extracting data from websites. Python libraries like BeautifulSoup and Scrapy make this process easier.


# Web Scraping Example
import requests
from bs4 import BeautifulSoup

url = "https://www.example.com"
response = requests.get(url)
soup = BeautifulSoup(response.text, "html.parser")

print(soup.title.text)

Python Data Visualization

Data visualization in Python is often done using libraries like Matplotlib, Seaborn, and Plotly.


# Data Visualization Example
import matplotlib.pyplot as plt

x = [1, 2, 3, 4]
y = [10, 20, 25, 30]

plt.plot(x, y)
plt.xlabel("X axis")
plt.ylabel("Y axis")
plt.title("Simple Line Plot")
plt.show()

Python for Data Science

Python is widely used in data science for data manipulation, analysis, and machine learning.


# Data Science Example
import pandas as pd

data = {"Name": ["Alice", "Bob", "Charlie"], "Age": [25, 30, 35]}
df = pd.DataFrame(data)
print(df)
