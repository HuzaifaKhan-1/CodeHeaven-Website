Introduction to C++

C++ is a powerful general-purpose programming language. It is widely used for systems software, game development, and more.

C++ Installation

C++ can be installed using various compilers like GCC, Clang, or IDEs like Visual Studio.

  1. Install a C++ compiler (e.g., GCC).
  2. Set up an IDE or text editor of your choice (e.g., Visual Studio, Code::Blocks).
  3. Write and compile your first C++ program.

C++ Syntax

C++ syntax is influenced by the C language but introduces object-oriented features.

C++ Data Types

C++ supports various data types such as integers, floating-point numbers, characters, and more.

  • int: Integer numbers
  • float: Floating-point numbers
  • char: Single character
  • string: Textual data (requires `#include `)

Object-Oriented Programming (OOP) Exercises

Practice your understanding of OOP concepts with these exercises.

  • Exercise 1: Create a Simple Class

    Define a class Person with attributes name and age. Implement a method introduce that prints a message introducing the person.

    #include <iostream>
        
        class Person {
        public:
            std::string name;
            int age;
        
            void introduce() {

        };

            person.age = 30;
            person.introduce();

        }

  • Exercise 2: Implement Inheritance

            return 0;

  • Exercise 3: Create a Class with Constructor and Destructor

  • Exercise 4: Implement Polymorphism

    Using the Shape class as a base class with a virtual method draw, create derived classes Circle and Square. Override the draw method in each derived class.

    #include <iostream>

        };
        
        class Circle : public Shape {
        public:

                std::cout << "Drawing a circle" << std::endl;

  • Exercise 8: Demonstrate Composition

    Implement a class Library that contains a collection of Book objects. Each Book has attributes title and author.

    #include <iostream>
        #include <vector>
        
        class Book {
        private:
            std::string title;
            std::string author;
        
        public:
            Book(const std::string& t, const std::string& a) : title(t), author(a) {}
        
            void display() {
                std::cout << "Title: " << title << ", Author: " << author << std::endl;
            }
        };
        
        class Library {
        private:
            std::vector books;
        
        public:
            void addBook(const Book& book) {
                books.push_back(book);
            }
        
            void displayBooks() {
                for (const auto& book : books) {
                    book.display();
                }
            }
        };
        
        int main() {
            Library library;
            library.addBook(Book("1984", "George Orwell"));
            library.addBook(Book("To Kill a Mockingbird", "Harper Lee"));
            library.displayBooks();
            return 0;
        }

  • Exercise 10: Implement Friend Classes

    Create a class Box with private attributes length, width, and height. Implement a friend class BoxPrinter that can access and print these attributes.

    #include <iostream>
        
        class BoxPrinter;
        
        class Box {
        private:

        };

        }

  • Exercise 11: Implement Operator Overloading

    Create a class Complex to represent complex numbers. Overload the + operator to add two complex numbers.

    #include <iostream>
        
        class Complex {
        private:
            double real;
            double imag;
        
        public:
            Complex(double r, double i) : real(r), imag(i) {}
        
            Complex operator+(const Complex& other) {
                return Complex(real + other.real, imag + other.imag);
            }
        
            void display() {
                std::cout << real << " + " << imag << "i" << std::endl;
            }
        };
        
        int main() {
            Complex c1(3.0, 2.0);
            Complex c2(1.5, 4.5);
            Complex c3 = c1 + c2;
            c3.display();
            return 0;
        }

C++ Operators

C++ supports various operators for arithmetic, comparison, logical operations, and more.

  • Arithmetic Operators: +, -, *, /, %
  • Comparison Operators: ==, !=, >, <, >=, <=
  • Logical Operators: &&, ||, !

Control Flow in C++

C++ supports control flow using `if`, `else`, and loops like `for` and `while`.


// Control Flow Example
int age = 18;
if (age >= 18) {
    cout << "You are an adult." << endl;
} else {
    cout << "You are a minor." << endl;
}

C++ Functions

Functions in C++ are defined using a return type, a name, and a parameter list.


// Function Example
int add(int a, int b) {
    return a + b;
}

int result = add(5, 10);
cout << result << endl;

Classes & Objects in C++

C++ supports object-oriented programming with classes and objects.


// Classes & Objects Example
class MyClass {
public:
    int myNumber;
    string myString;
};

int main() {
    MyClass myObj;
    myObj.myNumber = 15;
    myObj.myString = "Hello CodeHeaven";
    cout << myObj.myNumber << " " << myObj.myString << endl;
    return 0;
}

C++ Loops

Loops in C++ include `for`, `while`, and `do-while` loops.


// For Loop Example
for (int i = 0; i < 5; i++) {
    cout << i << endl;
}

C++ Arrays

Arrays in C++ are used to store multiple values of the same type.


// Array Example
int numbers[5] = {1, 2, 3, 4, 5};
cout << numbers[0] << endl; // Outputs 1

C++ Pointers

Pointers are variables that store the memory address of another variable.


// Pointer Example
int myVar = 10;
int* ptr = &myVar;
cout << *ptr << endl; // Outputs 10

File Handling in C++

C++ supports file handling with libraries like `fstream`.


// File Handling Example
#include 
using namespace std;

int main() {
    ofstream MyFile("example.txt");
    MyFile << "Hello CodeHeaven!";
    MyFile.close();
    return 0;
